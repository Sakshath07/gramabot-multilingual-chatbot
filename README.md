# GramaBot – Multilingual Government Scheme Assistant

GramaBot is an AI-powered chatbot that helps users access government welfare scheme details in multiple languages such as Kannada, Hindi, and English. It provides eligibility, benefits, required documents, PDF downloads, and QR codes linking to official portals. The system is designed to help rural users easily understand government schemes.

## Features
- Supports multiple languages
- Scheme details with eligibility and documents
- PDF and QR code generation
- Simple and accessible web interface

## Tech Stack
Flask • Python • HTML • CSS • JavaScript
